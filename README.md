# Memori-Log
Ruby on railsを用いて簡単な日記アプリを制作中。

## **現在実装しようとしている機能**  
最小限の機能を優先し、基本の投稿・閲覧機能を完成させる。  

1. **投稿機能**  
   - **タイトル・画像（任意）・コメントを記録** できる  
   - **投稿の編集・削除が可能**  

2. **投稿一覧表示（新しい順）**  
   - **ホーム画面に投稿を新しい順で表示** する  

3. **アルバムページ（画像のみ一覧表示）**  
   - 投稿された日記の **画像のみを一覧表示**  

4. **データの保存方法（Active Storage）**  
   - **画像は Active Storage を使用して保存**
   - 画像なしの投稿も許可
